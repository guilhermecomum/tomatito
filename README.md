This is a simple pomodoro Timer

Install all dependencies
```
$ npm install
```

Start the server
```
$ npm start
```

#### Usage
* [Access the web server](http://localhost:8080)
* To start the countdown, click on start button.
* You can reset the countdown, clicking on reset button
